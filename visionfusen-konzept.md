## Alleinstellungsmerkmal

> **Das Internet ohne den Rotz.**
> **Echte Menschen. Echte Gespräche.**
> **Vertrauen statt Algorithmen.**

Das Internet ist kaputt. 90% dessen, was du online siehst, ist Rauschen – Trolle, Bots, Spam, Wegwerf-Accounts. Niemand steht für das, was er sagt.

Bei Visionfusen ist das anders:

- Jedes Mitglied wurde eingeladen – oder hat sich beworben
- Jede Identität ist verifiziert
- Wer stört, fliegt raus
- Das Ergebnis: Gespräche mit echten Menschen

## Identitätsprinzip: Verifiziert, nicht anonym

### Drei Arten von Identität

| | Anonym | Pseudonym | Klarname |
|---|---|---|---|
| Beispiel | @j123170569 | "Leselampe" | Steven Noack |
| Geschichte | Keine | Jahrelang | Lebenslang |
| Reputation | Keine | Aufgebaut | Öffentlich |
| Konsequenz | Keine | Pseudonym verbrannt | Volle |
| **Bei Visionfusen** | ❌ Nein | ✓ Ja | ✓ Ja |

### Das Prinzip

> **Eine Person = Eine Identität = Verantwortung**
>
> Ob Klarname oder Pseudonym – spielt keine Rolle.
> Wichtig: Konsistenz, Reputation, Konsequenzen.

### Bedingungen für Pseudonyme

| Regel | Warum |
|---|---|
| Hafenmeister kennt echte Identität | Verantwortung |
| Bürge kennt echte Identität | Haftung |
| Eine Identität pro Person | Kein Wechseln |
| Langfristig | Keine Wegwerf-Accounts |

## Einladungs- & Moderationssystem

### Grundprinzip

```
Visionfusen ist kein offenes Netzwerk.
Es ist ein kuratierter Heimathafen.

Eintritt: Durch Einladung oder Bewerbung
Austritt: Freiwillig oder durch Ausschluss
```

### Ebenen der Mitgliedschaft

| Ebene | Rechte | Wie erreicht |
|---|---|---|
| **Gast** | Lesen, Landing Page | Jeder |
| **Anwärter** | Profil, eingeschränkt posten | Bewerbung eingereicht |
| **Mitglied** | Voll: Posten, Zappen, Chat | Bestätigt durch Einladung oder Bewährung |
| **Vertrauensperson** | Kann andere einladen | Nach 30 Tagen + 3 Empfehlungen |
| **Hafenmeister** | Alles + Moderation | Steven |

### Wege zur Mitgliedschaft

**Weg A: Einladung (bevorzugt)**

```
Vertrauensperson lädt ein
        ↓
Eingeladener bekommt Link mit Token
        ↓
Registrierung mit Email
        ↓
Sofort Mitglied
        ↓
Einladender ist "Bürge"
```

**Weg B: Bewerbung (offen)**

```
Interessent füllt Formular aus
├── Name (oder Pseudonym)
├── Email
├── "Warum Visionfusen?"
├── Optional: Website/Social
        ↓
Hafenmeister prüft (manuell)
        ↓
Annahme → Mitglied
Ablehnung → Freundliche Email
```

### Regeln für Vertrauenspersonen

| Regel | Warum |
|---|---|
| Max. 3 offene Einladungen gleichzeitig | Verhindert Masseneinladung |
| Neue Einladung erst nach 7 Tagen | Entschleunigung |
| Bürge haftet für Eingeladenen | Skin in the game |
| Bei Spam: Einladerechte entzogen | Selbstregulierung |

### Bürgen-Mechanismus

```
Anna lädt Bob ein
        ↓
Bob wird Mitglied
        ↓
Bob spammt
        ↓
Bob wird entfernt
        ↓
Anna bekommt Warnung
        ↓
Bei 2. Mal: Anna verliert Einladerechte für 90 Tage
        ↓
Bei 3. Mal: Anna verliert Einladerechte dauerhaft
```

### Bewährungssystem für Anwärter

```
Tag 1-7: Anwärter
├── Kann Profil ausfüllen
├── Kann 1 Post schreiben (in Warteschlange)
├── Kann lesen, aber nicht zappen
├── Kein Wallet, keine Sats
        ↓
Nach 7 Tagen ODER Empfehlung von 2 Mitgliedern:
        ↓
Wird Mitglied
├── Volle Rechte
├── 150 Starter-Sats
├── Wallet freigeschaltet
```

### Moderationsaktionen

| Aktion | Effekt | Reversibel? |
|---|---|---|
| **Verwarnen** | Private Nachricht | - |
| **Einschränken** | Kann lesen, nicht posten | ✓ |
| **Pausieren** | Account temporär deaktiviert | ✓ |
| **Entfernen** | Account gelöscht, NIP-05 weg | ✓ (aber Aufwand) |
| **Bannen** | Blacklist, kann nicht zurück | Nur manuell |

### Gründe für Entfernung

| Grund | Aktion |
|---|---|
| Spam | Sofort entfernen |
| Beleidigung/Hass | Warnung → Entfernen |
| Falsche Identität | Entfernen |
| Betrug | Bannen |
| Inaktiv >6 Monate | Pausieren (nicht löschen) |
| "Passt nicht" | Gespräch → ggf. Entfernen |

### Schutz vor Spam-Accounts

**Technische Maßnahmen:**

| Maßnahme | Wie |
|---|---|
| Email-Verifizierung | Bestätigungslink |
| Wegwerf-Email blocken | Liste von Domains |
| Rate Limiting | Max. 1 Account pro Email |
| IP-Tracking | Warnung bei vielen Accounts von einer IP |
| Captcha | Bei Bewerbung (nicht bei Einladung) |

**Ökonomische Maßnahmen:**

| Maßnahme | Effekt |
|---|---|
| Starter-Sats erst nach 7 Tagen | Spammer müssen warten |
| Auszahlung erst ab 1000 Sats | Lohnt sich nicht für 150 Sats |
| Sats nur intern nutzbar | Können nicht abgezogen werden |

```

   └── Liest Landing Page, Logbuch

2. INTERESSE
   └── Klickt "Mitmachen"
   └── Sieht: "Einladung oder Bewerbung"

4. ANWÄRTER (wenn Bewerbung)
   └── 7 Tage eingeschränkt
   └── Oder: 2 Mitglieder empfehlen → sofort Mitglied

5. MITGLIED
   └── Volle Rechte
   └── 150 Starter-Sats
   └── name@visionfusen.org

6. VERTRAUENSPERSON (nach 30 Tagen + Empfehlungen)
```
